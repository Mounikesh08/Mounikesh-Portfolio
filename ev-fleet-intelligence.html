EV Fleet Intelligence Platform – Battery Intelligence & FSD Scenario Analytics

A production-style Applied AI platform for electric vehicle fleet monitoring, battery anomaly detection, State-of-Health forecasting, FSD-style scenario risk mining, GenAI/RAG-style risk explanation, and live fleet dashboard visualization.

View GitHub Live Demo FastAPI Docs

Problem

Electric vehicle fleets generate large volumes of battery telemetry and driving-event data, but engineering teams need a reliable way to identify battery degradation, abnormal operating patterns, risky autonomous-driving scenarios, driver interventions, and near-collision events before they create safety, maintenance, or uptime issues.

Solution

Built and deployed an end-to-end EV Fleet Intelligence Platform with two modules: Battery Intelligence and FSD Scenario Intelligence. The Battery module detects abnormal battery behavior, forecasts future State-of-Health, and explains risk signals. The FSD module mines autonomous-driving style events, predicts scenario risk probability, and recommends whether the event should be prioritized for fleet review or model training data mining.

Architecture

  • Synthetic EV battery telemetry and FSD-style driving-event generation
  • Battery health feature engineering for voltage, temperature, impedance, capacity, SOC, and SOH behavior
  • FSD scenario feature engineering for lead distance, relative speed, lane offset, braking, weather, and road context
  • Battery anomaly detection using Isolation Forest
  • Battery State-of-Health forecasting using Random Forest Regressor
  • FSD scenario risk classification using Random Forest Classifier
  • GenAI/RAG-style explanation layer for engineer-friendly battery risk summaries
  • FastAPI backend with prediction, forecasting, explanation, health, and FSD risk endpoints
  • Streamlit dashboard for executive overview, battery intelligence, FSD scenario intelligence, and model operations
  • MLflow experiment tracking, Joblib model persistence, API smoke tests, GitHub version control, and Render deployment

Tech Stack

Python FastAPI Streamlit scikit-learn MLflow Pandas NumPy Plotly Pydantic Joblib Docker Render

Key Features

  • Battery anomaly detection with high-temperature, impedance, voltage-drop, and current-behavior risk signals
  • Short-horizon battery State-of-Health forecasting for 1-day, 3-day, and 7-day prediction windows
  • FSD-style scenario risk mining using speed, lead-vehicle distance, relative speed, lane offset, braking, pedestrians, red lights, and weather
  • GenAI/RAG-style battery explanation that converts model outputs into maintenance-ready engineering recommendations
  • Live Streamlit dashboard connected to deployed FastAPI backend through environment-based API configuration
  • API smoke tests validating health, anomaly prediction, forecasting, explanation, and FSD risk endpoints

Impact

  • Created a production-style AI system that demonstrates end-to-end ML engineering from data generation to live deployment
  • Combined EV battery intelligence with FSD scenario analytics to support predictive maintenance and autonomous-driving data review
  • Deployed a live demo with FastAPI backend, Streamlit frontend, GitHub repository, MLflow tracking, and API validation workflow
